# First Wave Growth Website

A modern, responsive website for First Wave Growth, an AI company specialising in automation services and SaaS products.

## Features

- Mobile-first responsive design
- Modern UI with gradient effects
- Smooth animations and micro-interactions
- Optimised for fast loading
- Cross-browser compatible

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with custom properties
- `script.js` - JavaScript for interactive elements
- `Resources/` - Contains SVG logos and images

## Customization

### Colors
The main colors are defined as CSS variables in the `:root` selector in `styles.css`.

You can easily change these colors by updating the variables.

### Contact Information
Update the email in the footer section with your actual contact details.

## Browser Support
This website is compatible with all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Performance Optimization
- Optimised animations for smooth performance
- Minimal JavaScript for essential functionality
